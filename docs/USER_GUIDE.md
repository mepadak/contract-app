## 시작하기

### 최초 접속

1. 앱에 처음 접속하면 4자리 PIN 설정 화면이 나타납니다.
2. 원하는 PIN 번호를 입력하고 확인을 누릅니다.
3. PIN이 설정되면 메인 채팅 화면으로 이동합니다.

### 로그인

- 앱에 접속할 때마다 설정한 PIN을 입력합니다.
- 5회 연속 실패 시 일시적으로 접근이 제한됩니다.

## 채팅 인터페이스

채팅 인터페이스는 자연어로 계약 업무를 처리하는 핵심 기능입니다.

### 사용 방법

1. 화면 하단의 입력창에 원하는 작업을 자연어로 입력합니다.
2. 전송 버튼을 누르거나 Enter 키를 누릅니다.
3. AI가 요청을 이해하고 적절한 작업을 수행합니다.

### 퀵 액션 버튼

자주 사용하는 명령어는 퀵 액션 버튼으로 빠르게 실행할 수 있습니다:

- **전체 목록**: 모든 계약 목록 조회
- **대시보드**: 현황 대시보드 열기
- **새 계약**: 계약 등록 시작
- **도움말**: 사용 가능한 명령어 안내

### 명령어 예시

#### 계약 등록

```
"서버 유지보수 용역 5천만원 일반경쟁 등록해줘"
"네트워크 장비 물품 3억 제한경쟁 등록"
"보안 솔루션 용역 수의계약 등록"
```

**필수 정보:**
- 계약명 (예: 서버 유지보수)
- 계약 종류 (용역, 물품, 공사)
- 계약 방법 (일반경쟁, 수의계약 등)

**선택 정보:**
- 금액 (예: 5천만원, 3억)
- 요청부서
- 마감일

#### 계약 조회

```
"전체 목록 보여줘"
"진행중인 계약"
"지연된 계약 목록"
"용역 계약만 보여줘"
"C26-001 상세"
"서버 계약 찾아줘"
```

#### 단계 변경

```
"C26-001 공고중으로 변경"
"서버 계약 개찰완료로 진행"
"그거 계약준비로 변경해줘"
```

**참고:** "그거", "그 계약"은 마지막으로 언급한 계약을 의미합니다.

#### 상태 변경

```
"C26-001 지연으로 변경"
"서버 계약 대기로 변경"
```

#### 메모 추가

```
"C26-001에 메모: 규격서 검토 완료"
"서버 계약에 메모 추가: 담당자 연락처 010-1234-5678"
```

#### 계약 삭제

```
"C26-001 삭제"
"서버 유지보수 계약 삭제해줘"
```

**주의:** 삭제된 계약은 목록에서 제외되지만 데이터는 유지됩니다 (Soft Delete).

### 금액 입력 방식

다양한 형식으로 금액을 입력할 수 있습니다:

| 입력 | 인식 금액 |
|------|----------|
| 5천만원 | 50,000,000원 |
| 5000만원 | 50,000,000원 |
| 5억 | 500,000,000원 |
| 1.5억 | 150,000,000원 |
| 50000000 | 50,000,000원 |

## 계약 관리

### 계약 목록 화면

화면 하단 네비게이션에서 "목록" 아이콘을 탭하면 계약 목록을 볼 수 있습니다.

#### 검색

- 상단 검색창에 키워드를 입력하면 계약명, 요청부서, 계약상대방에서 검색합니다.

#### 필터

필터 버튼을 탭하면 다음 조건으로 필터링할 수 있습니다:

- **상태**: 시작 전, 진행 중, 대기, 지연, 완료
- **종류**: 물품(구매), 물품(제조), 용역, 공사
- **방법**: 일반경쟁, 제한경쟁, 지명경쟁, 공개수의, 비공개수의

#### 계약 카드

각 계약 카드에는 다음 정보가 표시됩니다:

- 계약 ID (예: C26-001)
- 계약명
- 상태 배지
- 종류, 방법, 금액
- 현재 단계 및 진행률
- 마감일 (D-Day)

### 계약 상세

계약 카드를 탭하면 상세 정보를 볼 수 있습니다:

- **기본 정보**: ID, 계약명, 종류, 방법, 금액, 요청부서, 계약상대방
- **진행 현황**: 현재 단계, 진행률 바
- **메모**: 등록된 메모 목록
- **변경 이력**: 모든 변경 기록

## 대시보드

화면 하단 네비게이션에서 "대시보드" 아이콘을 탭하면 전체 현황을 볼 수 있습니다.

### 예산 현황

- **총 예산**: 연간 배정된 총 예산
- **배정 예산**: 진행 중인 계약 금액 합계
- **집행 예산**: 완료된 계약 금액 합계
- **잔여 예산**: 총 예산 - 배정 예산
- **집행률**: 집행 예산 / 총 예산 × 100

### 상태별 현황

각 상태별 계약 건수와 금액을 막대 차트로 표시합니다.

### 주의 필요 계약

다음 조건에 해당하는 계약이 알림 목록에 표시됩니다:

- 🔴 **위험**: 마감일 D-3 이내, 상태 "지연"
- 🟡 **주의**: 마감일 D-7 이내, 상태 "대기" 7일 이상

주의 계약을 탭하면 해당 계약 상세로 이동합니다.

## 설정

화면 하단 네비게이션에서 "설정" 아이콘을 탭합니다.

### 연간 예산 설정

1. "연간 예산" 메뉴를 탭합니다.
2. 새 예산 금액을 입력합니다 (예: 50억, 5000000000).
3. "저장" 버튼을 탭합니다.

### 로그아웃

설정 화면에서 "로그아웃"을 탭하면 세션이 종료되고 로그인 화면으로 이동합니다.

## 자주 묻는 질문

### Q: PIN을 잊어버렸어요

A: 현재 버전에서는 PIN 재설정 기능이 없습니다. 관리자에게 문의하여 데이터베이스에서 PIN을 초기화해야 합니다.

### Q: 삭제한 계약을 복구할 수 있나요?

A: 삭제된 계약은 데이터베이스에 유지되지만 현재 앱에서 복구 기능은 제공하지 않습니다. 관리자에게 문의하세요.

### Q: 오프라인에서 사용할 수 있나요?

A: 오프라인에서는 사용이 제한됩니다. 인터넷 연결이 끊어지면 화면 상단에 알림이 표시됩니다.

### Q: AI가 잘못 이해했을 때는?

A: 계약 생성, 수정, 삭제 등 중요한 작업은 항상 미리보기가 표시됩니다. 내용을 확인한 후 "확인" 버튼을 누르기 전에 "취소"를 선택할 수 있습니다.

### Q: 지원하는 브라우저는?

A: Chrome, Safari, Samsung Internet에서 최적화되어 있습니다. 모바일 환경 (360px ~ 428px)에서 가장 잘 동작합니다.
